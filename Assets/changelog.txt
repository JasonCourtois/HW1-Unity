AI Usage:

I did use Copilot (using Claude Sonnet 4.5) to debug an issue with my color changing script.
The color changing was occurring way too fast. This happened because I didn't save a reference of the old color before making the Color.lerp() call.
This meant that as the color was changing to the new color, I was lerping from the in-between color to the new desired color every frame. This caused a rapid change in color.
Copilot caught this, so I then made a change to store the old material color in a variable before calling Color.lerp().

I also used Copilot to explain what the "clamped numeric defaults" requirement meant for the camera control script.

I also used the Unity Pro Builder package to modify the geometry of the tree parts without affecting the transforms of the children components.
Before using ProBuilder, scaling the tree trunk would mess with the scale of the children transforms, so using Pro Builder allowed me to modify the geometry without touching the scale in the trunk's transform.
Once I used ProBuilder to make the geometry I wanted, I removed the ProBuilder scripts that way no errors would occur when importing the asset package.
